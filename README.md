# makelp

Vite + React + TypeScript + TailwindCSSを使用したWebアプリケーション

## 技術スタック

- **Vite** - 高速なビルドツール
- **React** - UIライブラリ
- **TypeScript** - 型安全な開発
- **TailwindCSS** - ユーティリティファーストCSSフレームワーク
- **React Router** - ルーティング管理

## インストール

```bash
npm install
```

## 開発環境の起動

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## プレビュー

```bash
npm run preview
```

## コード検証

```bash
npm run lint
```

## ディレクトリ構成

```
makelp/
├── src/           # ソースコード
├── public/        # 静的ファイル
├── dist/          # ビルド出力
└── ...           # 設定ファイル
```
